The `<PreviewOnly>` component is used to render content only when [preview mode](/docs/4.x/api/composables/use-preview-mode) is enabled.

::note
Unlike [`<DevOnly>`](/docs/4.x/api/components/dev-only), `<PreviewOnly>` is controlled at runtime and its content is not removed from production builds.
::

Use [`usePreviewMode`](/docs/4.x/api/composables/use-preview-mode) to configure how preview mode is enabled. `<PreviewOnly>` only renders based on that configured preview state.

```vue [app/pages/articles/[slug].vue]
<template>
  <div>
    <PreviewOnly>
      <DraftToolbar />

      <template #fallback>
        <PublishedToolbar />
      </template>
    </PreviewOnly>
  </div>
</template>
```

## Slots

- `#fallback`: specify content to render when preview mode is not enabled.
