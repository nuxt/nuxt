# B7016

The custom SPA loading template file specified in your configuration could not be found at the given path.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- The `spaLoadingTemplate` path in your config points to a non-existent file
- The file was moved or deleted without updating the configuration

## Resolution

Fix the file path in your `nuxt.config.ts` to point to an existing template file, or remove the option to use the default SPA loading template:

```ts
export default defineNuxtConfig({
  spaLoadingTemplate: './path/to/existing/template.html',
})
```
