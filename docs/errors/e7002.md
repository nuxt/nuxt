# E7002

Nuxt was unable to load the payload for a given route. This typically occurs with static site generation when a route's payload file is missing.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Payload file not found on the server
- Network error preventing payload retrieval
- Route not included in prerendering for static sites

## Resolution

Ensure the route is included in prerendering for static sites. Check your `nitro.prerender.routes` configuration or add `routeRules` for the affected routes.
