# E7002

Nuxt could not load the payload for a route. With static site generation this usually means the route's payload file was not generated because the route was not prerendered.

## Resolution

Ensure the affected route is prerendered. Add it to `nitro.prerender.routes`, or enable crawling/`prerender: true` via `routeRules` in `nuxt.config.ts`:

```ts
export default defineNuxtConfig({
  routeRules: {
    '/my-route': { prerender: true },
  },
})
```
