# B4004

Two page files resolve to the same route name. This often happens with pages in different directories that map to the same path. Route names must be unique for navigation and route resolution to work correctly.

## Resolution

Give one of the conflicting pages a custom name with `definePageMeta`:

```ts
definePageMeta({
  name: 'my-custom-route-name',
})
```
