# E4003

The content passed as a slot to `<NuxtLayout>` has multiple root nodes. `<NuxtLayout>` requires its slot content to have a single root element for transitions and rendering to work correctly.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Multiple elements passed as children of `<NuxtLayout>`
- Stray text or comments alongside the main content element

## Resolution

Wrap the content inside `<NuxtLayout>` in a single root element.
