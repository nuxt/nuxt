# B8013

A module was skipped because its compatibility constraints are not met by the current Nuxt version or environment. Unlike [B8004](/docs/api/advanced/errors), the build continues without the module.

## Resolution

```bash
npx nuxi upgrade        # update Nuxt
npm install <module>@latest  # or update the module
```
