# B8013

A module has been disabled because its compatibility constraints are not met by the current Nuxt version or environment.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- The module's compatibility constraints are not met by the current Nuxt version
- The module requires features or APIs not available in this version of Nuxt

## Resolution

Update Nuxt or the module to compatible versions:

```bash
# Update Nuxt
npx nuxi upgrade

# Or update the module
npm install <module-name>@latest
```
