# B4010

A middleware file could not be resolved to a name. This happens when a middleware file is named `index`, which is ignored for middleware naming purposes.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Middleware file named index which is ignored for naming
- Middleware file without a resolvable name

## Resolution

Rename the middleware file to a descriptive name, such as `auth.ts` or `admin.ts`.
