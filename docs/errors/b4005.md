# B4005

`definePageMeta()` was called with a variable or expression instead of an object literal. The metadata must be an object literal so it can be statically analyzed at build time.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Passing variable or expression instead of object literal
- Using a pre-defined object reference

## Resolution

Pass an object literal directly to `definePageMeta()`:

```ts
// Wrong
const meta = { title: 'My Page' }
definePageMeta(meta)

// Correct
definePageMeta({ title: 'My Page' })
```
