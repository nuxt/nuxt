# B8008

A falsy (null, undefined, or empty) template value was passed to a template registration function. A valid template object is required.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- A falsy template value was passed to `addTemplate()` or similar function
- A variable holding the template resolved to `null` or `undefined`

## Resolution

Ensure you pass a valid template object:

```ts
addTemplate({
  filename: 'my-template.mjs',
  getContents: () => 'export default {}',
})
```
