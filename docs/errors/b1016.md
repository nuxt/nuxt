# B1016

Nuxt could not load `@nuxt/webpack-builder`. This happens when the builder is set to `webpack` in your Nuxt configuration but the package is not installed.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- @nuxt/webpack-builder not installed but builder set to webpack
- Incompatible version of @nuxt/webpack-builder

## Resolution

Install the webpack builder package:

```bash
npm install -D @nuxt/webpack-builder
```
