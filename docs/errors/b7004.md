# B7004

The webpack entry point could not be found. The server build requires a valid entry configuration to produce the server bundle.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- Incorrect entry configuration in the webpack settings
- The entry file was moved or deleted

## Resolution

Check your webpack entry option and ensure it points to a valid file. If you have custom webpack configuration, verify that the entry path is correct.
