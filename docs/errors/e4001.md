# E4001

Nuxt detected that a layout was selected that does not exist in your project. This usually means the layout name passed to `definePageMeta` or `setPageLayout` does not match any file in the `layouts/` directory.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Typo in layout name
- Layout file deleted or renamed

## Resolution

Check that the layout name matches a file in your `layouts/` directory. For example, if you use `definePageMeta({ layout: 'custom' })`, ensure `layouts/custom.vue` exists.
