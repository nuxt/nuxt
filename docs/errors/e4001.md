# E4001

A layout was selected that does not exist in your project. The name passed to `definePageMeta` or `setPageLayout` does not match any file in the `layouts/` directory — usually a typo, or a renamed/deleted layout file.

## Resolution

Ensure the layout name matches a file in `layouts/`. For example, `definePageMeta({ layout: 'custom' })` requires `layouts/custom.vue` to exist.
