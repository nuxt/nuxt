# B1015

Nuxt fell back to the `chokidar-granular` file watcher because `@parcel/watcher` is not installed or failed to load on your platform. Watching still works, but `@parcel/watcher` is faster.

## Resolution

Install `@parcel/watcher` for better file-watching performance:

```bash
npm install -D @parcel/watcher
```
