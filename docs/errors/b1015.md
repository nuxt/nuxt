# B1015

Nuxt is falling back to the `chokidar-granular` file watcher because `@parcel/watcher` is not installed. While `chokidar-granular` works, `@parcel/watcher` provides better performance for file watching.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- @parcel/watcher not installed
- @parcel/watcher failed to load on current platform

## Resolution

Install `@parcel/watcher` for better file watching performance:

```bash
npm install -D @parcel/watcher
```
