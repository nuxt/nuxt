# B2008

A plugin's `dependsOn` property references one or more plugins that are not registered. The dependency cannot be resolved because the referenced plugins do not exist.

::note
**This is a build error (logged).** This error is logged during the build process. The build may continue, but the issue should be investigated.
::

## Common causes

- dependsOn references plugins that don't exist
- Typo in plugin name within dependsOn array

## Resolution

Register the missing plugins or remove the dependency from the `dependsOn` array. Verify that plugin names match exactly.
