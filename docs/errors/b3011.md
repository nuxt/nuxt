# B3011

Two component files resolve to the same component name. Only one will be used, which may lead to unexpected behavior.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Two component files resolve to same name
- Components in different directories with identical names

## Resolution

Rename one of the conflicting components or adjust the `components.dirs` prefix configuration to differentiate them.
