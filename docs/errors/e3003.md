# E3003

`useFetch()` or `useAsyncData()` was called after the component had already mounted (e.g. inside an event handler, `watch` callback, or `setTimeout`). These composables are designed for the setup phase and SSR only.

## Resolution

Reserve `useFetch()` and `useAsyncData()` for setup. After the component has mounted, use `$fetch()` directly to fetch data.
