# E6003

The `<Style>` component received non-string content. The `<Style>` component expects a CSS string as its slot content.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Non-string content passed to the `<Style>` component
- Using HTML elements or components inside `<Style>`

## Resolution

Only pass a CSS string to the `<Style>` component:

```vue
<Style>body { color: red }</Style>
```
