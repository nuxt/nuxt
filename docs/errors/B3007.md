# B3007

The `nuxt-client` attribute is being used in island components without enabling the `selectiveClient` option. This feature requires explicit opt-in.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Using nuxt-client in islands without enabling selectiveClient
- Missing experimental configuration for selective client hydration

## Resolution

Set `experimental.componentIslands.selectiveClient` to `true` in your `nuxt.config`:

```ts
export default defineNuxtConfig({
  experimental: {
    componentIslands: {
      selectiveClient: true
    }
  }
})
```
