# E7008

The `fn` parameter passed to `callOnce` is not a function. `callOnce` expects a function to execute.

::note
**This is a runtime error.** Your application will throw when this condition is encountered in both development and production.
::

## Common causes

- Passing a non-function value as the `fn` parameter to `callOnce`
- Accidentally passing the result of a function call instead of the function itself

## Resolution

Pass a function to `callOnce`:

```ts
// Incorrect
callOnce('key', someValue)

// Correct
callOnce('key', () => { /* ... */ })
```
