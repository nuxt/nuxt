# B8014

A module's setup function took longer than expected to complete. Slow module setup can significantly impact build and development server startup times.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Module setup took too long due to expensive operations
- Network requests or heavy computations during module setup

## Resolution

Optimize the module's setup logic. Consider:

- Deferring expensive operations to build hooks instead of setup
- Caching results of expensive computations
- Avoiding synchronous blocking operations
- Moving network requests to later build stages
