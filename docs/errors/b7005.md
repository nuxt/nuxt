# B7005

`vite.build.rollupOptions.input` is missing the `entry` key needed for the client build — usually because a custom `input` overwrote Nuxt's default.

## Resolution

Keep an `entry` key (or pass a string pointing to the client entry):

```ts
export default defineNuxtConfig({
  vite: {
    build: {
      rollupOptions: {
        input: {
          entry: '/path/to/entry',
        },
      },
    },
  },
})
```
