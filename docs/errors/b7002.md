# B7002

Vite's pre-bundled dependency cache (`optimizeDeps`) is stale, usually after a dependency change. This can cause slow dev startup or unexpected behavior from outdated pre-bundles.

## Resolution

Clear the Vite cache and restart the dev server:

```bash
rm -rf node_modules/.vite
```
