# B1004

Nuxt attempted to automatically install a required package but the installation failed. This can happen when Nuxt detects a missing dependency and tries to install it on-the-fly.

::note
**This is a build error (logged).** This error is logged during the build process. The build may continue, but the issue should be investigated.
::

## Common causes

- Network error
- Package registry unavailable
- Permissions issue

## Resolution

Install the package manually using your package manager (e.g., `npm install`, `pnpm add`, or `yarn add`).
