# B5004

Nuxt detected a `vite.config` or `webpack.config` file alongside your `nuxt.config`. External bundler configuration files are not supported in Nuxt projects as Nuxt manages the bundler configuration internally.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- A `vite.config.ts` or `webpack.config.js` file exists in the project root alongside `nuxt.config`
- The file was left over from a migration or copied from a non-Nuxt project

## Resolution

Move your bundler configuration into `nuxt.config.ts` using the `vite` or `webpack` config keys:

```ts
export default defineNuxtConfig({
  vite: {
    // Your Vite config here
  }
})
```

Then remove the external config file.
