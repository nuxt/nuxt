# B2004

A plugin file was found but it has no content. Empty plugin files serve no purpose and should either be populated or removed.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Empty plugin file
- Plugin file with only whitespace or comments

## Resolution

Add plugin content to the file or remove the empty plugin file from the `plugins/` directory.
