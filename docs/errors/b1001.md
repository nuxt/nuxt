# B1001

The build process failed to compile a template file. This typically occurs when a template registered by a module or Nuxt itself contains invalid syntax or references missing variables during compilation.

::note
**This is a build error (logged).** This error is logged during the build process. The build may continue, but the issue should be investigated.
::

## Common causes

- Syntax error in template
- Invalid template content

## Resolution

Check the template file mentioned in the error message for syntax errors. If the template is provided by a module, ensure the module is up to date. If it is a custom template, verify the syntax and ensure all required variables are available.
