# B1019

An unknown component mode was encountered during the component transform phase. This is an internal Nuxt bug that should not occur under normal usage.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- Internal Nuxt bug
- Unexpected component mode value in transform plugin

## Resolution

Report this issue on the [Nuxt GitHub repository](https://github.com/nuxt/nuxt/issues) with reproduction steps and the full error message.
