# B4001

A page file was found but it has no content. Empty page files cannot be rendered and should either be populated with content or removed.

::note
**This is a build error (logged).** This error is logged during the build process. The build may continue, but the issue should be investigated.
::

## Common causes

- Empty page file
- Page file with only whitespace

## Resolution

Add content to the page file or remove it from the `pages/` directory.
