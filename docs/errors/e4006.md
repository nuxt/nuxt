# E4006

SSR rendering failed inside a `<NuxtClientFallback>` component, causing it to display its fallback content. This warning indicates that the fallback mechanism is working as designed.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- SSR rendering failed inside `<NuxtClientFallback>`
- A child component threw an error during server-side rendering

## Resolution

This is informational — the `<NuxtClientFallback>` component is working as designed. If you do not want the fallback to be triggered, fix the underlying SSR error in the child component.
