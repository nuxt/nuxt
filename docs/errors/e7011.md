# E7011

The key passed to `useAsyncData` is not a string. `useAsyncData` requires a string key to identify the data and enable deduplication and caching.

::note
**This is a runtime error.** Your application will throw when this condition is encountered in both development and production.
::

## Common causes

- Passing a non-string value as the key to `useAsyncData`
- Using a variable that resolves to a non-string type

## Resolution

Pass a string key to `useAsyncData`:

```ts
useAsyncData('myKey', () => $fetch('/api/data'))
```
