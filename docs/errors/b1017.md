# B1017

Nuxt failed to load the configured builder package, which is responsible for bundling your application. This usually means the package is missing or its version is incompatible with your Nuxt version.

## Resolution

Install the builder set in your Nuxt config: `@nuxt/vite-builder` for Vite (default) or `@nuxt/webpack-builder` for webpack, and ensure its version is compatible.
