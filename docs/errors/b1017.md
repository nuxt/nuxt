# B1017

Nuxt failed to load the configured builder package. The builder is responsible for bundling your application and must be installed and compatible with your Nuxt version.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- Builder package not installed or incompatible
- Version mismatch between Nuxt and the builder package

## Resolution

Install the builder package specified in your Nuxt configuration. For Vite (default), ensure `@nuxt/vite-builder` is available. For webpack, install `@nuxt/webpack-builder`.
