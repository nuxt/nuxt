# B1008

An internal error occurred where a keyed function factory could not be found. This is an internal Nuxt bug that should not occur under normal circumstances.

::note
**This is a build error (logged).** This error is logged during the build process. The build may continue, but the issue should be investigated.
::

## Common causes

- Internal Nuxt bug
- Unexpected state in keyed function factory resolution

## Resolution

Report this issue on the [Nuxt GitHub repository](https://github.com/nuxt/nuxt/issues) with reproduction steps and the full error message.
