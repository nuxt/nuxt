# B1007

A compiler plugin's `afterScan` hook threw an error. The `afterScan` hook runs after all files have been scanned and is used by plugins to perform post-scan processing.

::note
**This is a build error (logged).** This error is logged during the build process. The build may continue, but the issue should be investigated.
::

## Common causes

- Plugin afterScan hook threw an error
- Bug in a compiler plugin's post-scan logic

## Resolution

Check the plugin mentioned in the error for errors in its `afterScan` hook. If the plugin is third-party, update it or report the issue to the maintainers.
