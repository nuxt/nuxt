# B6001

Your code is importing directly from `@unhead/vue` instead of using the `#imports` alias. In Nuxt, head composables should be imported from `#imports` to ensure proper integration with the Nuxt auto-import system.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Direct `import { ... } from '@unhead/vue'` statements in your code
- Copy-pasting code from non-Nuxt examples that import from `@unhead/vue` directly

## Resolution

Replace direct imports from `@unhead/vue` with imports from `#imports`, or rely on Nuxt's auto-import system:

```diff
- import { useHead } from '@unhead/vue'
+ import { useHead } from '#imports'
```

Or simply remove the import and let Nuxt auto-import handle it.
