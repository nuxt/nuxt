# B6001

Head composables (like `useHead`) are being imported directly from `@unhead/vue`. In Nuxt these should come from `#imports` (or be left to auto-imports) so they integrate with Nuxt's head management.

## Resolution

```diff
- import { useHead } from '@unhead/vue'
+ import { useHead } from '#imports'
```

Or remove the import entirely and let auto-imports resolve it.
