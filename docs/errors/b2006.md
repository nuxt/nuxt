# B2006

An error occurred while parsing a plugin file. The file could not be analyzed for plugin metadata due to syntax errors.

::note
**This is a build error (logged).** This error is logged during the build process. The build may continue, but the issue should be investigated.
::

## Common causes

- Syntax error in plugin file
- Invalid JavaScript or TypeScript in the plugin

## Resolution

Fix the syntax errors in the plugin file. Check for missing brackets, unclosed strings, or other parsing issues.
