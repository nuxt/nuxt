# E2006

`addRouteMiddleware()` was called without a valid middleware function. The middleware argument was `undefined` or `null`.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Passing `undefined` or `null` as the middleware argument to `addRouteMiddleware()`
- A variable that was expected to hold a middleware function is not defined

## Resolution

Ensure a valid middleware function is passed to `addRouteMiddleware()`.
