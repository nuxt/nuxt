# B1009

Two keyed functions with the same name were registered from different sources. Keyed function names must be unique across the build to avoid conflicts.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Two keyed functions with the same name from different sources
- Multiple modules registering functions with identical names

## Resolution

Rename one of the conflicting functions to ensure each keyed function has a unique name.
