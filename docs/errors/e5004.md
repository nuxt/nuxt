# E5004

You attempted to set a response header (e.g. via `useResponseHeader()`) on the client. Response headers can only be set during server-side rendering.

## Resolution

Guard the call so it only runs on the server:

```ts
if (import.meta.server) {
  useResponseHeader('X-Custom-Header', 'value')
}
```
