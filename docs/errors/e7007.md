# E7007

`useState` expects its `init` argument to be a *factory function* that returns the initial value, not the value itself. This avoids sharing the same object reference across requests on the server.

## Resolution

Wrap the initial value in a function:

```ts
// Incorrect
useState('key', myValue)

// Correct
useState('key', () => myValue)
```
