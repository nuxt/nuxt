# E4012

An error occurred inside a server (island) component during rendering. This warning surfaces the error details to help you debug the underlying issue in the component.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Error inside a server/island component
- Unhandled exception during component rendering or data fetching

## Resolution

Check the full error details in the console and fix the underlying component error. The error message will include specifics about what went wrong inside the island component.
