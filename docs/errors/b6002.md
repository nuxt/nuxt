# B6002

A custom composable shares its name with a Nuxt built-in (or a module-registered) auto-import. With two auto-imports under the same name, Nuxt cannot reliably pick which to use.

## Resolution

Rename the conflicting composable so it no longer collides with the built-in:

```ts
// useHead.ts conflicts with the Nuxt built-in → rename it
export function useCustomHead () {
  // ...
}
```
