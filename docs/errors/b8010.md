# B8010

A template was registered without either a `getContents` function or a `src` path. At least one of these must be provided for Nuxt to generate the template content.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- A template object is missing both `getContents` and `src` properties
- The template was partially configured

## Resolution

Provide either a `getContents` function or a `src` path:

```ts
// Option 1: Using getContents
addTemplate({
  filename: 'my-template.mjs',
  getContents: () => 'export default {}',
})

// Option 2: Using src
addTemplate({
  filename: 'my-template.mjs',
  src: resolve('./runtime/template.ejs'),
})
```
