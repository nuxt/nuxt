# E2011

Nuxt failed to preload a route component chunk. This can happen when the chunk file is unavailable, typically due to a stale deployment or network issue.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Route component chunk failed to load from the server
- Component file was moved or deleted after the manifest was generated

## Resolution

This is usually transient. Consider implementing the `app:chunkError` hook to handle chunk loading failures gracefully (e.g., triggering a hard reload).
