# B8009

The template source file specified by the `src` property does not exist at the given path.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- The template `src` path points to a non-existent file
- The file was moved or deleted without updating the template registration

## Resolution

Verify the template file path exists. Use `createResolver` to resolve paths relative to your module:

```ts
const { resolve } = createResolver(import.meta.url)

addTemplate({
  src: resolve('./runtime/template.ejs'),
  filename: 'my-template.mjs',
})
```
