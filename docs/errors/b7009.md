# B7009

Decorators are being used in your project but the required Babel packages for decorator support are not installed.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- TypeScript decorators are used without the required Babel transform packages
- The decorator support packages were not installed during project setup

## Resolution

Install the listed packages shown in the warning message. Typically this includes Babel decorator transform plugins:

```bash
npm install -D @babel/plugin-proposal-decorators
```
