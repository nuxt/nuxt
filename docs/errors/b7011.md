# B7011

A PostCSS plugin could not be imported during the webpack build. The plugin is either not installed or cannot be resolved.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- A PostCSS plugin referenced in your configuration is not installed
- The plugin package cannot be resolved from the project's `node_modules`

## Resolution

Install the missing PostCSS plugin:

```bash
npm install -D <postcss-plugin-name>
```

Verify that the plugin name in your PostCSS configuration matches the installed package name.
