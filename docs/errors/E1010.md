# E1010

A Vue error occurred during rendering, but the user agent indicates the request is from a bot. Nuxt skips rendering the error page for bot user agents to provide more useful content for crawlers.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- A Vue error occurred but the user agent indicates a bot or crawler
- Search engine bots encountering rendering errors

## Resolution

This is informational — bot user agents intentionally skip error page rendering. Fix the underlying Vue error to resolve this warning for all users.
