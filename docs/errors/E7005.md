# E7005

A cookie is being set with an already-expired `maxAge` or `expires` value. The cookie will be immediately discarded by the browser.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Setting a cookie with a negative or zero `maxAge`
- Setting a cookie with an `expires` date in the past

## Resolution

Use a positive `maxAge` value or a future `expires` date when setting cookies:

```ts
const cookie = useCookie('name', {
  maxAge: 60 * 60 * 24 * 7, // 7 days
})
```
