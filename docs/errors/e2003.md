# E2003

`abortNavigation()` was called outside of a route middleware context. This function is only valid inside route middleware, where it can intercept and cancel the current navigation.

::note
**This is a dev-only error.** This error only occurs in development mode and is removed from production builds.
::

## Common causes

- Calling `abortNavigation()` from a component, plugin, or composable instead of from route middleware
- Using `abortNavigation()` in a callback that has lost the middleware context

## Resolution

Move the `abortNavigation()` call into a route middleware function. It can only be used within the middleware handler.
