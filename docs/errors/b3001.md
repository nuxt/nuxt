# B3001

A directory listed in `components.dirs` configuration does not exist. Nuxt scans these directories for auto-imported components, but the specified path could not be found.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Directory listed in components.dirs doesn't exist
- Typo in the directory path configuration

## Resolution

Create the missing directory or remove it from the `components.dirs` configuration in `nuxt.config`.
