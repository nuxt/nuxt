# B1014

An error occurred while checking for external configuration files such as `vite.config` or `webpack.config`. This check is non-critical and the build will continue.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Error checking for vite.config or webpack.config files
- File system permission issue in project root

## Resolution

This is usually harmless — the check will be skipped. If you rely on an external config file, verify it exists and is accessible.
