# E3005

The `execute` function from `useAsyncData()` was passed directly as a `watch` callback (`watch(someRef, execute)`). `watch` passes the new value as the first argument, which changes how `execute` runs.

## Resolution

Wrap it in an inline function so no arguments are forwarded:

```ts
watch(someRef, () => execute())
```
