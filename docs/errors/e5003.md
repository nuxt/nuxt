# E5003

Nuxt encountered an error while trying to match route rules from the app manifest. This typically indicates an issue with your route rules configuration.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Invalid `routeRules` configuration in `nuxt.config.ts`
- Malformed route patterns or rule values

## Resolution

Check your `routeRules` configuration in `nuxt.config.ts` for any syntax errors or invalid patterns.
