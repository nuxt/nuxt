# B1002

The build process encountered an error while reading a template source file. The template's `src` path could not be read from disk.

::note
**This is a build error (logged).** This error is logged during the build process. The build may continue, but the issue should be investigated.
::

## Common causes

- Template source file missing or unreadable
- File permissions issue

## Resolution

Verify that the template `src` path exists and is readable. If the template is provided by a module, ensure the module is correctly installed and its files are intact.
