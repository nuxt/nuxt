# B8007

A type template filename does not end with `.d.ts`. Type templates must have the `.d.ts` extension to be recognized as TypeScript declaration files.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- A type template filename doesn't end with `.d.ts`
- The filename extension was incorrectly specified when adding a type template

## Resolution

Ensure the type template filename ends with `.d.ts`:

```ts
addTypeTemplate({
  filename: 'types/my-module.d.ts', // Must end with .d.ts
  getContents: () => '// type declarations'
})
```
