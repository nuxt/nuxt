# B1011

The build cache encountered an I/O error while reading a file. This indicates a file system-level problem rather than a concurrency issue.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Disk I/O error
- File system permissions

## Resolution

Check file system health and permissions. Ensure the disk is not full and the build process has read access to all project files.
