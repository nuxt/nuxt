# B7008

JSX is being used in your project but the `@vitejs/plugin-vue-jsx` plugin is not installed. This plugin is required for Vue JSX/TSX support in Vite.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- JSX/TSX files are used in the project without the Vue JSX plugin
- The plugin was not installed when setting up JSX support

## Resolution

Install the Vue JSX plugin:

```bash
npm install -D @vitejs/plugin-vue-jsx
```
