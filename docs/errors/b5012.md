# B5012

A required package needs to be installed. Nuxt has provided a suggested install command to resolve this issue.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- A package needs to be installed to enable a feature
- This is an actionable hint (useful for AI agents and developers) with the exact command to run

## Resolution

Run the suggested install command shown in the warning message. For example:

```bash
npm install <suggested-package>
```
