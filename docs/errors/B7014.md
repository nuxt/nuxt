# B7014

The webpack build failed due to compilation errors. The build output should contain specific error messages indicating what went wrong.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- webpack compilation errors (syntax errors, missing modules, type errors)
- Incompatible webpack loader or plugin configurations

## Resolution

Fix the errors shown in the build output. Common fixes include:

- Resolve import paths that cannot be found
- Fix syntax errors in your source files
- Ensure all required loaders and plugins are installed
- Check for version incompatibilities between webpack plugins
