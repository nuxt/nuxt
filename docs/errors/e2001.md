# E2001

`navigateTo()` was called with an external URL but without the `{ external: true }` option. Nuxt requires explicit opt-in for external navigation to prevent accidental redirects away from your application.

::note
**This is a runtime error.** Your application will throw when this condition is encountered in both development and production.
::

## Common causes

- Passing a full URL (e.g., `https://example.com`) to `navigateTo()` without the `external` option
- Redirecting to an external site from middleware without setting `external: true`

## Resolution

Add `{ external: true }` as the second argument to `navigateTo()` when navigating to external URLs:

```ts
navigateTo('https://example.com', { external: true })
```
