# B5002

Nuxt attempted to auto-install `@nuxt/webpack-builder` but the installation failed. This typically happens when the builder is set to webpack but the required package cannot be resolved.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Auto-install of `@nuxt/webpack-builder` failed due to network or permission issues
- The package manager is unable to resolve the package

## Resolution

Install the webpack builder manually:

```bash
npm install -D @nuxt/webpack-builder
```

Or switch to the Vite builder (default) by removing or changing the `builder` option in your `nuxt.config.ts`.
