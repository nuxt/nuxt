# B5002

Your `builder` is set to webpack, but Nuxt could not auto-install `@nuxt/webpack-builder` — usually due to network or permission issues, or the package manager failing to resolve the package.

## Resolution

Install the webpack builder manually:

```bash
npm install -D @nuxt/webpack-builder
```

Or switch back to the default Vite builder by removing the `builder` option from your `nuxt.config`.
