# B4003

`definePageMeta()` was called more than once in a single page file. Only one call to `definePageMeta()` is allowed per page.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- Calling definePageMeta() more than once in a page
- Duplicate definePageMeta() from copy-paste

## Resolution

Consolidate all page metadata into a single `definePageMeta()` call.
