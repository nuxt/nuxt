# B3004

A component was used that belongs to an uninstalled Nuxt module. The component cannot be resolved until the required module is installed.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- Component from uninstalled Nuxt module
- Module was removed but component references remain

## Resolution

Install the required module using the Nuxt CLI:

```bash
npx nuxi module add <module-name>
```
