# E7004

`definePayloadReviver` was called outside of the expected context. It must be called within a Nuxt plugin that is registered early in the plugin lifecycle.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- `definePayloadReviver` called outside of an early Nuxt plugin
- Called in a component or composable instead of a plugin

## Resolution

Move the `definePayloadReviver` call to a Nuxt plugin that is registered early in the lifecycle. Ensure the plugin runs before the payload is processed.
