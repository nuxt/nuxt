# E3002

`useFetch()` failed to generate a hash for the request body. Nuxt uses body hashing to create unique cache keys for requests. This warning indicates the body could not be serialized for hashing.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- The request body is not a plain object, `ArrayBuffer`, or `FormData`
- The body contains circular references that prevent serialization

## Resolution

Ensure the body is serializable. For custom body types, provide a custom `key` option to `useFetch()` to avoid automatic body hashing.
