# E6001

A head composable (such as `useHead()`) was called without an active Unhead instance. This happens when head composables are called outside of the expected Nuxt lifecycle contexts.

::note
**This is a runtime error.** Your application will throw when this condition is encountered in both development and production.
::

## Common causes

- Calling `useHead()` outside of `setup()`, a plugin, or middleware
- Calling head composables in a detached async context where the Nuxt instance is no longer available

## Resolution

Call head composables inside a component `setup()` function, a Nuxt plugin, or route middleware where the Nuxt context is available.
