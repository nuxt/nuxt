# E1007

A compiler-hint helper such as `definePageMeta()` was called at runtime. These helpers are compile-time macros that are transformed by the Nuxt build pipeline and should never execute at runtime.

::note
**This is a runtime error.** Your application will throw when this condition is encountered in both development and production.
::

## Common causes

- Calling `definePageMeta()` inside a composable instead of directly in a page's `<script setup>`
- Calling compiler macros in non-page components

## Resolution

Only call these helpers at the top level of a page component's `<script setup>` block. They cannot be used dynamically or inside composables.
