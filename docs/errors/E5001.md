# E5001

You attempted to use an app manifest API, but the app manifest feature is not enabled in your Nuxt configuration.

::note
**This is a runtime error.** Your application will throw when this condition is encountered in both development and production.
::

## Common causes

- Calling manifest APIs without enabling the experimental feature
- Using a module or composable that depends on the app manifest

## Resolution

Enable the app manifest feature in your `nuxt.config.ts`:

```ts
export default defineNuxtConfig({
  experimental: {
    appManifest: true
  }
})
```
