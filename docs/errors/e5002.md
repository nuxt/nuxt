# E5002

Nuxt failed to fetch the app manifest from the server. The manifest is used for features like route rules and prerendering hints.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Network error when fetching the app manifest
- Build output missing or corrupted

## Resolution

Check your network connectivity and ensure the build output is intact. If the issue persists, try rebuilding your application.
