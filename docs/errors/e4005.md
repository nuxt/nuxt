# E4005

A server (island) component was found with multiple root elements. Server components must have exactly one root element for Nuxt to correctly serialize and hydrate them on the client.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Island component with multiple root elements
- Root-level HTML comments in the component template

## Resolution

Ensure the server component has exactly one root element. Remove any root-level HTML comments or wrap the content in a single container element.
