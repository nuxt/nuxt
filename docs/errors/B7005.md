# B7005

The `rollupOptions.input` configuration is missing the required `entry` key for the client build. Vite needs a valid client entry point to build the application.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- Missing `entry` key in `rollupOptions.input` for the client build
- Custom `rollupOptions.input` configuration that overwrites the default entry

## Resolution

Ensure `rollupOptions.input` has an `entry` key or is a string pointing to the client entry:

```ts
export default defineNuxtConfig({
  vite: {
    build: {
      rollupOptions: {
        input: {
          entry: '/path/to/entry',
        },
      },
    },
  },
})
```
