# B4006

Non-serializable values were used in `definePageMeta()`. Page metadata is extracted at build time and must contain only serializable values.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Non-serializable values in definePageMeta
- Functions, class instances, or symbols in page metadata

## Resolution

Use only serializable values in `definePageMeta()`: strings, numbers, booleans, arrays, and plain objects. Move non-serializable logic to middleware or composables.
