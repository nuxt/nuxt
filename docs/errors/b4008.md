# B4008

Server pages are being used with `ssr: false`. Server pages render on the server, so they need either SSR enabled or the `componentIslands` experimental feature.

## Resolution

Enable `experimental.componentIslands` in your `nuxt.config`:

```ts
export default defineNuxtConfig({
  experimental: {
    componentIslands: true,
  },
})
```
