# B8005

Nuxt cannot determine its version from the provided Nuxt instance. This usually indicates that an invalid Nuxt instance was passed to a compatibility check function.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- An invalid Nuxt instance was passed to a compatibility check
- The Nuxt instance object is malformed or missing version information

## Resolution

Ensure you are passing a valid Nuxt instance. If you are writing a module, use `useNuxt()` to get the correct instance:

```ts
import { useNuxt } from '@nuxt/kit'

const nuxt = useNuxt()
```
