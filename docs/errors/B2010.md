# B2010

Nuxt could not statically analyze the plugin metadata because it contains complex expressions that cannot be evaluated at build time.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Complex expressions in plugin metadata that can't be statically analyzed
- Dynamic values or function calls in plugin options

## Resolution

Simplify plugin metadata to use static values. Avoid function calls, ternary expressions, or variable references in the metadata object.
