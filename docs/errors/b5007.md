# B5007

The webpack target is not set to `"node"` for the server build. Nuxt's server bundle must target Node.js to function correctly.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- webpack target not set to `"node"` for the server build
- A custom webpack configuration overrode the default target

## Resolution

Ensure the webpack target is set to `"node"` for the server build. Avoid overriding the target in your server-side webpack configuration.
