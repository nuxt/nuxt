# B1012

A cache file path was detected as unsafe and skipped. This can happen when a cache entry contains path traversal sequences or other unsafe characters that could write files outside the intended cache directory.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Cache file path contains path traversal or unsafe characters
- Corrupted or tampered build cache

## Resolution

Clean the build cache directory by removing the `.nuxt` folder and rebuilding.
