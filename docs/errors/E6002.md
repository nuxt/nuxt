# E6002

The `<Title>` component received multiple children or non-string content. The `<Title>` component expects a single text string as its slot content.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Multiple children or non-string content passed to the `<Title>` component
- Using HTML elements or components inside `<Title>`

## Resolution

Use a single string inside the `<Title>` component:

```vue
<Title>My Page Title</Title>
```
