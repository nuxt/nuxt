# B4008

Server pages are being used with `ssr: false` in the Nuxt configuration. Server pages require SSR or the `componentIslands` experimental feature to function.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Using server pages with ssr disabled
- Conflicting SSR and server page configuration

## Resolution

Set `experimental.componentIslands` to `true` in your `nuxt.config`:

```ts
export default defineNuxtConfig({
  experimental: {
    componentIslands: true
  }
})
```
