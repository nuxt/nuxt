# B8018

An error occurred while importing a module. The module was found but could not be imported, typically due to dependency issues.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- The module has unresolved dependency issues
- A transitive dependency is missing or incompatible
- The module has syntax errors or runtime import failures

## Resolution

Check that all of the module's dependencies are installed:

```bash
# Reinstall all dependencies
rm -rf node_modules
npm install
```

If the issue persists, check the module's documentation for required peer dependencies.
