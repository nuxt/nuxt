# B8018

A module was resolved but threw while being imported — typically a missing or incompatible (transitive) dependency, or an error in the module's own code.

## Resolution

Reinstall dependencies and check the module's required peer dependencies:

```bash
rm -rf node_modules && npm install
```

If it persists, the failure is inside the module itself — check its docs or report it to the author.
