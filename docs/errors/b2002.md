# B2002

The plugin metadata object contains spread elements or computed property keys, which cannot be statically analyzed at build time.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- Using spread elements or computed property keys in plugin metadata
- Dynamic property names in the options object

## Resolution

Use static properties instead of spread elements or computed keys in the plugin metadata object.
