# B4009

A layout file could not be resolved to a name. This happens when a layout file is named `index`, which is ignored for layout naming purposes.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Layout file named index which is ignored for layout naming
- Layout file without a resolvable name

## Resolution

Rename the layout file to a non-index name, such as `default.vue` or `custom.vue`.
