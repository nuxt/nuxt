# E7010

The key passed to `callOnce` is not a string. `callOnce` requires a string key to track whether the function has already been executed.

::note
**This is a runtime error.** Your application will throw when this condition is encountered in both development and production.
::

## Common causes

- Passing a non-string value as the key to `callOnce`
- Using a variable that is not of type string

## Resolution

Pass a string key to `callOnce`:

```ts
callOnce('myKey', () => { /* ... */ })
```
