# E2009

A route middleware threw an error, but no `router.onError()` handler was registered to catch it. Without an error handler, middleware errors may go unhandled.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- A middleware throws an error but no `router.onError()` handler is registered
- Missing global error handling for navigation failures

## Resolution

Register an error handler with `router.onError()` in a Nuxt plugin to catch and handle middleware errors gracefully.
