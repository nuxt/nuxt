# B8014

A module's `setup` function took longer than expected, slowing build and dev-server startup. This is a warning — the build continues.

## Resolution

Move expensive work out of `setup`:

- Defer heavy operations to later build hooks instead of running them eagerly in `setup`.
- Cache results of expensive computations.
- Avoid synchronous blocking work and move network requests to later stages.
