# B7015

Your project uses full-static output without payload extraction. Extracting the payload separates page data from the HTML, reducing initial page size for statically generated sites.

## Resolution

```ts
export default defineNuxtConfig({
  experimental: {
    payloadExtraction: true, // or 'client'
  },
})
```
