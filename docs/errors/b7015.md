# B7015

Your project uses full-static output but payload extraction is not enabled. Payload extraction allows Nuxt to separate the page data from the HTML, reducing the initial page size for statically generated sites.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Full-static output is configured without payload extraction
- The `experimental.payloadExtraction` option is not set

## Resolution

Enable payload extraction in your `nuxt.config.ts`:

```ts
export default defineNuxtConfig({
  experimental: {
    payloadExtraction: true, // or 'client'
  },
})
```
