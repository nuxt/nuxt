# E4010

Conflicting props were passed to `<NuxtLink>`. For example, both `to` and `href` were provided, which creates ambiguity about the intended navigation target.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Conflicting props passed to `<NuxtLink>` (e.g., `to` and `href` together)
- Programmatically generated props that include redundant navigation properties

## Resolution

Use only one of the conflicting props. For internal navigation, use `to`. For external links, use `href` or `to` with an external URL.
