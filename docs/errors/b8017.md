# B8017

Nuxt could not load a module because the package is not installed. The module is referenced in the configuration but cannot be resolved.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- The module package is not installed in the project
- The module name is misspelled in the configuration

## Resolution

Install the module package:

```bash
npm install <module-name>
```

Verify the module name in your `nuxt.config.ts` matches the installed package name.
