# B4012

An incremental route update failed during development (HMR). Nuxt automatically performs a full rebuild when this happens, so the issue is usually recovered from automatically.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- HMR route update failed during development
- Transient file system issue during hot reload

## Resolution

This is usually transient — Nuxt performs a full rebuild automatically. If it persists, restart the development server.
