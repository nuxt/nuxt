# B2001

The second argument to `defineNuxtPlugin` must be an object literal. A non-object value was passed as plugin metadata.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- Passing non-object as second argument to defineNuxtPlugin
- Using a variable reference instead of an inline object

## Resolution

Use an object literal for plugin options when calling `defineNuxtPlugin()`.
