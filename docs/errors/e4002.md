# E4002

A layout component was found with multiple root nodes. Nuxt layouts must have a single root element to ensure proper transitions and rendering.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Layout with multiple root elements
- Stray HTML comments at the root level of the layout

## Resolution

Wrap your layout content in a single root element, such as a `<div>`.
