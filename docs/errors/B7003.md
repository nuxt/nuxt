# B7003

The server bundle must have a single entry point, but `optimization.splitChunks` is enabled in the server webpack configuration. Splitting chunks in the server bundle is not supported and will cause the build to fail.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- `optimization.splitChunks` is enabled in the server webpack config
- A global webpack configuration was applied to both client and server builds

## Resolution

Remove `splitChunks` from the server webpack configuration. If you need chunk splitting, apply it only to the client build:

```ts
export default defineNuxtConfig({
  webpack: {
    optimization: {
      splitChunks: false // for server
    }
  }
})
```
