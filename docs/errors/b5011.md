# B5011

A Nuxt module package is not installed. Nuxt attempted to resolve a module but could not find the required package.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- A Nuxt module package is not installed in your project
- The module was referenced in `nuxt.config` but its package was never installed

## Resolution

Follow the install prompt shown in the warning, or install the package manually:

```bash
npm install <module-package>
```
