# E4014

`<NuxtPage>` is used in your application but no `pages/` directory exists. Nuxt needs a `pages/` directory with at least one page component for `<NuxtPage>` to render content.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- `<NuxtPage>` used but no `pages/` directory exists
- The `pages/` directory was accidentally deleted or renamed

## Resolution

Create a `pages/` directory in your project root with at least one page component (e.g., `pages/index.vue`).
