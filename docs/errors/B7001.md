# B7001

Bundle analysis was requested but `rollup-plugin-visualizer` is not installed. This plugin is required to generate bundle analysis reports.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- `rollup-plugin-visualizer` is not installed in the project
- Bundle analysis was enabled in the Nuxt config without installing the required dependency

## Resolution

Install `rollup-plugin-visualizer` as a dev dependency:

```bash
npm install -D rollup-plugin-visualizer
```
