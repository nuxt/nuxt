# E1006

`onPrehydrate()` was called at runtime without being processed by the Nuxt build pipeline. This composable requires compile-time transformation to work correctly and is only available on the server.

::note
**This is a runtime error.** Your application will throw when this condition is encountered in both development and production.
::

## Common causes

- Calling `onPrehydrate()` from an untranspiled dependency
- Using the composable in a library that is not included in the build transpile list

## Resolution

Add the library to `build.transpile` in `nuxt.config.ts` so the build pipeline can process the `onPrehydrate()` call.
