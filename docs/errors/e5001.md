# E5001

You called an app manifest API (directly or via a module/composable that depends on it), but the app manifest is an experimental feature that is disabled by default.

## Resolution

Enable it in `nuxt.config.ts`:

```ts
export default defineNuxtConfig({
  experimental: {
    appManifest: true,
  },
})
```
