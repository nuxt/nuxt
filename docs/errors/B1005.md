# B1005

A compiler plugin encountered an error while scanning a source file. The build pipeline uses plugins to analyze and transform files, and one of these plugins failed during the scan phase.

::note
**This is a build error (logged).** This error is logged during the build process. The build may continue, but the issue should be investigated.
::

## Common causes

- Syntax error in scanned file
- Plugin bug

## Resolution

Check the file mentioned in the error message for syntax errors. If the issue is in a third-party plugin, update it to the latest version or report the issue to the plugin maintainers.
