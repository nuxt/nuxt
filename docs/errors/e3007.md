# E3007

The Options API `asyncData()` hook returned a non-object value. The `asyncData()` hook must return a plain object whose properties are merged into the component's data.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Returning a primitive value (string, number, boolean) from the Options API `asyncData()` hook
- Returning an array instead of an object from `asyncData()`

## Resolution

Return a plain object from `asyncData()`. Consider migrating to `useAsyncData()` in the Composition API for more flexibility.
