# E1011

An error occurred while mounting the Nuxt application on the client side. This can happen during hydration or initial client-side render when the app fails to mount correctly.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Client-side app mounting fails during hydration or initial render
- A plugin or component error occurs during the mount phase

## Resolution

Check the full error details in the console. Common causes include hydration mismatches and missing dependencies. Ensure server-rendered HTML matches client-side expectations.
