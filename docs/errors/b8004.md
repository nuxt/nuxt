# B8004

A module's `compatibility` constraints are not satisfied by the installed Nuxt version.

## Resolution

Bring the two into a compatible range — upgrade Nuxt or the module:

```bash
npx nuxi upgrade        # update Nuxt
npm install <module>@latest  # or update the module
```
