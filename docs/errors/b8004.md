# B8004

A module has compatibility issues with the current version of Nuxt. The module requires a different version of Nuxt than what is installed.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- A module requires a different Nuxt version than what is installed
- The module's `compatibility` constraints do not match the current Nuxt version

## Resolution

Update either Nuxt or the module to compatible versions:

```bash
# Update Nuxt
npx nuxi upgrade

# Or update the module
npm install <module-name>@latest
```
