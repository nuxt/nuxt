# B2005

A plugin file was found but it does not have a default export. Nuxt plugins must export a default function, typically wrapped in `defineNuxtPlugin`.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Plugin file missing export default
- Accidentally using named export instead of default export

## Resolution

Add a default export to the plugin file:

```ts
export default defineNuxtPlugin(() => {})
```
