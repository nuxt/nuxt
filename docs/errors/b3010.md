# B3010

The component scanning process could not determine a file name for a component. This usually indicates an issue with the file extension or naming convention.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Component scanning could not determine file name
- File with unexpected or missing extension

## Resolution

Ensure the component file has a proper extension (e.g., `.vue`, `.ts`, `.tsx`) and follows Nuxt's component naming conventions.
