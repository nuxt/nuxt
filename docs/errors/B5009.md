# B5009

Nuxt is falling back to chokidar for schema file watching because `@parcel/watcher` is not available. While chokidar works, `@parcel/watcher` provides better performance for file watching.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- `@parcel/watcher` is not installed in your project
- `@parcel/watcher` failed to install due to native dependency compilation issues

## Resolution

Install `@parcel/watcher` for improved file watching performance:

```bash
npm install -D @parcel/watcher
```
