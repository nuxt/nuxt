# E4007

Your project has layout files in the `layouts/` directory, but the `<NuxtLayout>` component is not used in your `app.vue`. Without `<NuxtLayout>`, your layout files will have no effect.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Project has layout files but no `<NuxtLayout>` component in `app.vue`
- `<NuxtLayout>` was accidentally removed during refactoring

## Resolution

Add `<NuxtLayout>` to your `app.vue` wrapping `<NuxtPage>`:

```vue
<template>
  <NuxtLayout>
    <NuxtPage />
  </NuxtLayout>
</template>
```
