# B5003

You are using custom keys under the `runtimeConfig.app` namespace, which is reserved by Nuxt for internal use. Placing custom configuration here may conflict with Nuxt's internal runtime config values.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Custom keys defined under `runtimeConfig.app` in `nuxt.config`
- Accidentally nesting configuration under the reserved `app` namespace

## Resolution

Move your custom runtime config keys to `runtimeConfig.public` or a custom namespace:

```ts
export default defineNuxtConfig({
  runtimeConfig: {
    // Instead of runtimeConfig.app.myKey
    public: {
      myKey: 'value'
    }
  }
})
```
