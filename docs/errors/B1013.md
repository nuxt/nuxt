# B1013

The build process failed to restore a file from the build cache. This typically indicates a corrupted cache entry or a disk error during cache restoration.

::note
**This is a build error (logged).** This error is logged during the build process. The build may continue, but the issue should be investigated.
::

## Common causes

- Corrupted cache entry
- Disk error

## Resolution

Clear the build cache by removing the `.nuxt` folder and rebuild the project.
