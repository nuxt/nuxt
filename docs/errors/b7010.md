# B7010

Decorators are being used in the Nitro server context but the required Babel packages for decorator support are not installed.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- TypeScript decorators are used in Nitro server code without the required Babel transform packages
- The decorator support packages were not installed for the server-side build

## Resolution

Install the listed packages shown in the warning message:

```bash
npm install -D @babel/plugin-proposal-decorators
```
