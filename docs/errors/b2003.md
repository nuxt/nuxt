# B2003

The `dependsOn` property in plugin metadata must be an array of string literals. A non-array or array with non-string elements was provided.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- dependsOn is not an array of string literals
- Using variables or expressions in the dependsOn array

## Resolution

Use an array of string literals for the `dependsOn` property, e.g., `dependsOn: ['my-plugin', 'other-plugin']`.
