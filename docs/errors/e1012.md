# E1012

A `vue:setup` hook callback returned a Promise. These callbacks must be synchronous.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- A `vue:setup` hook callback is declared as `async` or returns a Promise

## Resolution

Remove `async` from your `vue:setup` hook callback, or move async logic into `onMounted` or a plugin.
