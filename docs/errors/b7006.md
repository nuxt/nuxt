# B7006

The `rollupOptions.input` configuration is missing the required `server` key for the server build. Vite needs a valid server entry point to build the server bundle.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- Missing `server` key in `rollupOptions.input` for the server build
- Custom `rollupOptions.input` configuration that overwrites the default server entry

## Resolution

Ensure `rollupOptions.input` has a `server` key:

```ts
export default defineNuxtConfig({
  vite: {
    build: {
      rollupOptions: {
        input: {
          server: '/path/to/server-entry',
        },
      },
    },
  },
})
```
