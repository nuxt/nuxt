# B7006

`vite.build.rollupOptions.input` is missing the `server` key needed for the server build — usually because a custom `input` overwrote Nuxt's default.

## Resolution

Keep a `server` key pointing to the server entry:

```ts
export default defineNuxtConfig({
  vite: {
    build: {
      rollupOptions: {
        input: {
          server: '/path/to/server-entry',
        },
      },
    },
  },
})
```
