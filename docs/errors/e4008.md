# E4008

A test component wrapper attempted to reference a file path outside of the project root directory. This is a security restriction to prevent accessing files outside the project boundary.

::note
**This is a runtime error.** Your application will throw when this condition is encountered in both development and production.
::

## Common causes

- Test component path references a file outside the project root
- Using relative paths (e.g., `../../`) that resolve outside the project directory

## Resolution

Ensure all component paths used in tests are within the project root directory. Use paths relative to the project root rather than paths that traverse above it.
