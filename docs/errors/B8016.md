# B8016

A module was resolved from its path or package name, but the resolved value is not a function. Nuxt modules must export a function as their default export.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- The module resolved but does not export a function
- The module's default export is an object or other non-callable value
- The module was not built correctly or is missing its main entry

## Resolution

Ensure the module exports a function as its default export:

```ts
// my-module.ts
export default defineNuxtModule({
  setup(options, nuxt) {
    // module logic
  }
})
```
