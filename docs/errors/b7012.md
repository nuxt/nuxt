# B7012

A ViteNode IPC message exceeded the buffer size limit. This happens when a module or response transferred between the dev server processes is too large — often an oversized data object or a recursive/infinite structure generating runaway output.

## Resolution

Shrink what crosses the process boundary: reduce the response size, stream or paginate large datasets, and check for infinite loops or circular data in your server code.
