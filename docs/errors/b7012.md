# B7012

The IPC message size has exceeded the buffer limit in ViteNode. This typically occurs when a server response or module is too large to be transferred between processes.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- An IPC message exceeds the buffer size limit
- A very large response or data object is being passed between processes
- An infinite loop or recursive data structure is generating excessive output

## Resolution

Reduce the response size or check for infinite loops in your server code. If you are loading large datasets, consider streaming or pagination approaches instead.
