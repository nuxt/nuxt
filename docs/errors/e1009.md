# E1009

The `vue:error` hook handler itself threw an error. This means the error handling code encountered its own failure, which can mask the original error.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- The `vue:error` hook handler itself throws an unhandled error
- Accessing undefined properties or calling failing functions inside the error hook

## Resolution

Add proper error handling inside your `vue:error` hook to prevent it from throwing. Use `try/catch` within the hook to handle potential failures gracefully.
