# NUXT_B2011

The `addPlugin()` kit utility was called without providing a `src` option. Every plugin must have a source file.

::note
**This is a build-time error.** The build will fail when this condition is encountered.
::

## Common causes

- `addPlugin()` called without a `src` option
- A plugin configuration object missing its `src` property

## Resolution

Provide a `src` path when calling `addPlugin()`:

```ts
addPlugin({ src: resolve('./runtime/plugin') })
```
