# B8003

`useNitro()` was called before Nitro has been initialized. Nitro is only available after the `ready` hook has been executed.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- Calling `useNitro()` before the `ready` hook
- Attempting to access Nitro during module setup before initialization is complete

## Resolution

Use `useNitro()` only after the `ready` hook:

```ts
export default defineNuxtModule({
  setup (options, nuxt) {
    nuxt.hook('ready', () => {
      const nitro = useNitro()
      // Nitro is now available
    })
  },
})
```
