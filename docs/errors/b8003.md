# B8003

`useNitro()` was called before Nitro was initialized. The Nitro instance only exists after the `ready` hook has run.

## Resolution

```ts
export default defineNuxtModule({
  setup (options, nuxt) {
    nuxt.hook('ready', () => {
      const nitro = useNitro()
    })
  },
})
```
