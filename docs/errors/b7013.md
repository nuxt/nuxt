# B7013

The ViteNode socket path is not properly configured. This is typically an internal configuration issue that prevents communication between the Vite dev server and the Node.js server process.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- ViteNode socket path is misconfigured
- An internal configuration error occurred during the build setup

## Resolution

This is usually an internal configuration error. Try clearing your build cache and restarting the dev server:

```bash
rm -rf .nuxt node_modules/.vite
npx nuxi dev
```

If the issue persists, it may be a bug — consider reporting it on GitHub.
