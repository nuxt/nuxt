# B2007

A plugin file exports a function that is not wrapped in `defineNuxtPlugin`. While this may work, wrapping plugins in `defineNuxtPlugin` enables type safety and plugin metadata support.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Plugin exports a function not wrapped in defineNuxtPlugin
- Legacy plugin format without defineNuxtPlugin wrapper

## Resolution

Wrap the plugin function with `defineNuxtPlugin`:

```ts
export default defineNuxtPlugin(() => {
  // plugin logic
})
```
