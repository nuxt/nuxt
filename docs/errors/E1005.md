# E1005

An error occurred during the Nuxt application initialization phase. This can happen when a plugin throws an error or when an `app:created` or `app:beforeMount` hook fails.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- A plugin throws an unhandled error during execution
- An error in `app:created` or `app:beforeMount` hooks

## Resolution

Check the full error details in the console. Wrap plugin logic in `try/catch` to handle errors gracefully.
