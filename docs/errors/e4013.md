# E4013

A `v-for` directive was used with a non-integer number as the range value. Vue's `v-for` range expects an integer to iterate correctly.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- `v-for` used with a non-integer number (e.g., `v-for="n in 3.5"`)
- A computed value returning a float used as a `v-for` range

## Resolution

Use `Math.floor()`, `Math.ceil()`, or `Math.round()` to convert the number to an integer before using it as a `v-for` range value.
