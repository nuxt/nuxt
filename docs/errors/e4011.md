# E4011

Your project has page files in the `pages/` directory, but you are using `<RouterView>` instead of `<NuxtPage>`. Nuxt requires `<NuxtPage>` to properly handle page routing, transitions, and keep-alive behavior.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Project has page files but uses `<RouterView>` instead of `<NuxtPage>`
- Migrating from a Vue Router project without updating the template

## Resolution

Replace `<RouterView>` with `<NuxtPage>` in your `app.vue`, or set `pages: false` in your `nuxt.config.ts` if you intend to manage routing manually.
