# E4011

Your project has files in `pages/`, but the template uses `<RouterView>` instead of `<NuxtPage>` — common when migrating from a plain Vue Router setup. Nuxt needs `<NuxtPage>` to handle routing, transitions, and keep-alive.

## Resolution

Replace `<RouterView>` with `<NuxtPage>` in `app.vue`. If you intend to manage routing manually, set `pages: false` in `nuxt.config.ts`.
