# B8011

A template was registered without a `filename` property. Every template must have a filename so Nuxt knows where to write the generated file.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- A template object is missing the `filename` property
- The filename was accidentally omitted during template configuration

## Resolution

Provide a `filename` property for the template:

```ts
addTemplate({
  filename: 'my-template.mjs',
  getContents: () => 'export default {}',
})
```
