# B5005

Nuxt was unable to load the schema file. This typically indicates that the schema file has syntax errors or incorrect exports.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Schema file has syntax errors
- Schema file has incorrect or missing exports
- The schema file cannot be resolved or parsed

## Resolution

Check your schema file for syntax errors and ensure it has valid exports. Verify that the file compiles correctly and exports the expected schema structure.
