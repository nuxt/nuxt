# B1018

Nuxt failed to load the server builder package. The server builder is responsible for building the server-side portion of your application.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- Server builder package not installed or incompatible
- Version mismatch between Nuxt and the server builder

## Resolution

Install the server builder package. Ensure it is compatible with your current Nuxt version.
