# B5010

One or more required packages are not installed in your project. Nuxt detected missing dependencies that are needed for the build to function correctly.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Required packages are not installed in the project
- Dependencies were removed or not included after cloning the repository

## Resolution

Install the missing packages as indicated in the warning message:

```bash
npm install <missing-package>
```
