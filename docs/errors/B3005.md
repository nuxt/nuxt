# B3005

Multiple lazy-hydration attributes were detected on the same component. Only one hydration strategy should be applied per component instance.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Multiple lazy-hydration attributes on same component
- Conflicting hydration directives

## Resolution

Use only one hydration strategy per component. Remove the extra hydration attributes and keep only the desired one.
