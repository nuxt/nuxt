# B3003

Standalone server components are being used without enabling the `componentIslands` experimental feature. This feature flag is required for standalone server component support.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Using standalone server components without the feature flag
- Missing experimental configuration

## Resolution

Set `experimental.componentIslands` to `true` in your `nuxt.config`:

```ts
export default defineNuxtConfig({
  experimental: {
    componentIslands: true
  }
})
```
