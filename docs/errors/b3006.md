# B3006

Lazy-hydration props were applied to a component that is not prefixed with `Lazy`. Hydration strategies only work with lazy-loaded components.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Hydration props on component not prefixed with Lazy
- Forgetting to use the Lazy prefix when adding hydration attributes

## Resolution

Either rename the component to use the `Lazy` prefix (e.g., `<LazyMyComponent>`) or remove the hydration props.
