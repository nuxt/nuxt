# E4004

A page component was found with multiple root nodes. Nuxt pages should have a single root element to ensure proper page transitions and rendering behavior.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Page component with multiple root elements
- Stray HTML comments or text nodes at the root level of the page

## Resolution

Wrap your page content in a single root element, such as a `<div>`.
