# B4002

The `await` keyword was used inside `definePageMeta()`. Page metadata must be synchronously defined because it is extracted at build time through static analysis.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- Using await inside definePageMeta()
- Async operations in page metadata definition

## Resolution

Move async logic outside of `definePageMeta()`. Page metadata must be statically analyzable at build time. Use middleware or `setup()` for async operations.
