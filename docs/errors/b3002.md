# B3002

You are using `.server.vue` components while SSR is disabled (`ssr: false`). Server components need either SSR or the `componentIslands` experimental feature to render.

## Resolution

Enable `componentIslands` so server components work without SSR:

```ts
export default defineNuxtConfig({
  experimental: {
    componentIslands: true,
  },
})
```
