# B5008

An invalid PostCSS order preset string was provided. The value specified for `postcss.order` does not match any known preset.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- Invalid `postcss.order` string in Nuxt configuration
- Typo in the preset name

## Resolution

Use a valid preset name for the `postcss.order` configuration option. Check the Nuxt documentation for available PostCSS order presets.
