# E1003

A runtime config key was accessed on the client side that is not available. Only keys defined under `runtimeConfig.public` are exposed to the client. The warning message will list the available client-side keys.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- Accessing server-only `runtimeConfig` keys on the client
- Typo in the config key name

## Resolution

Move the key to `runtimeConfig.public` in `nuxt.config.ts` if it needs client access. Keep sensitive keys server-only.
