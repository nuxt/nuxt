# B1006

A compiler plugin failed to read a source file during the build process. This typically happens when a file is removed or becomes inaccessible between the time it was discovered and when it was read.

::note
**This is a build error (logged).** This error is logged during the build process. The build may continue, but the issue should be investigated.
::

## Common causes

- File deleted during build
- File system error

## Resolution

Ensure the file exists and is readable. If files are being modified by an external process during the build, try running the build again.
