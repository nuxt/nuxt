# E2002

`navigateTo()` was called with a URL that uses a potentially dangerous protocol such as `javascript:`, `data:`, or `vbscript:`. Nuxt blocks these navigations to prevent XSS and other security vulnerabilities.

::note
**This is a runtime error.** Your application will throw when this condition is encountered in both development and production.
::

## Common causes

- User-supplied input passed directly to `navigateTo()` without sanitization
- A URL with `javascript:`, `data:`, or `vbscript:` protocol

## Resolution

Validate and sanitize all user-provided URLs before passing them to `navigateTo()`. Only allow `http:`, `https:`, or relative paths.
