# B1003

A template was registered without the required `src` or `getContents` property. Every template must provide one of these so that Nuxt knows how to generate its content.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- Template missing both src and getContents
- Misconfigured addTemplate() call in module

## Resolution

Provide either `src` or `getContents` when calling `addTemplate()`. If you are using a module that registers templates, ensure it is up to date.
