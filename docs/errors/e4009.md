# E4009

A `<NuxtLink>` was rendered inside another `<NuxtLink>`, producing nested `<a>` tags in the HTML output. Nested anchor elements are invalid HTML and can cause unexpected behavior in browsers.

::note
**This is a development warning.** This warning only appears in development mode and does not affect production builds.
::

## Common causes

- `<NuxtLink>` rendered inside another `<NuxtLink>`, producing nested `<a>` tags
- A parent component wraps content in a link, and a child component also contains a link

## Resolution

Use the `custom` prop on the outer `<NuxtLink>` to prevent it from rendering an `<a>` tag, or restructure your markup so that links are not nested.
