# E4009

A `<NuxtLink>` was rendered inside another `<NuxtLink>`, producing nested `<a>` tags. Nested anchors are invalid HTML and behave unpredictably in browsers — often caused by a parent component wrapping content in a link while a child also contains one.

## Resolution

Restructure your markup so links are not nested, or use the `custom` prop on the outer `<NuxtLink>` so it does not render its own `<a>` tag.
