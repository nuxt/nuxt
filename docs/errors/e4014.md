# E4014

`<NuxtPage>` is used but no `pages/` directory exists (it may have been deleted or renamed). `<NuxtPage>` needs a `pages/` directory with at least one page component to render content.

## Resolution

Create a `pages/` directory in your project root with at least one page, e.g. `pages/index.vue`.
