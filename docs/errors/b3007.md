# B3007

The `nuxt-client` attribute is used inside island components, but selective client hydration is not enabled. It must be opted into explicitly.

## Resolution

Enable `selectiveClient` under `componentIslands`:

```ts
export default defineNuxtConfig({
  experimental: {
    componentIslands: {
      selectiveClient: true,
    },
  },
})
```
