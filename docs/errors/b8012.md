# B8012

A Nuxt module function was called outside of the Nuxt module system context. Modules must be executed within the Nuxt lifecycle to have access to the required context.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- A module function was called directly without the Nuxt context
- The module was imported and invoked as a regular function instead of through Nuxt's module system

## Resolution

Only use modules within Nuxt's module system. Register modules in your `nuxt.config.ts`:

```ts
export default defineNuxtConfig({
  modules: [
    'my-module',
    // or inline module
    (options, nuxt) => { /* ... */ },
  ],
})
```
