# B3008

A component directory name differs only in case from the expected convention. This may cause issues on case-sensitive file systems and lead to unexpected component naming.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Directory name differs only in case from expected convention
- Case mismatch between file system and component references

## Resolution

Rename the directory to match the expected case convention for component directories.
