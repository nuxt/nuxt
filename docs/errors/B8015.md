# B8015

An invalid module type was passed to Nuxt. Modules must be either a function or a string (module path/package name).

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- An invalid module type (not a function or string) was passed in the `modules` array
- An object or other non-callable value was used where a module was expected

## Resolution

Pass a function or string module path in your `nuxt.config.ts`:

```ts
export default defineNuxtConfig({
  modules: [
    'my-module',              // string (package name)
    './modules/my-module.ts', // string (path)
    (options, nuxt) => {}     // function
  ]
})
```
