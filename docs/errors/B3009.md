# B3009

A component name starts with "Lazy" which conflicts with Nuxt's built-in lazy-loading prefix. Nuxt uses the `Lazy` prefix to automatically create lazy-loaded versions of components.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Component name starts with "Lazy" which conflicts with lazy-loading prefix
- Component file named with Lazy prefix (e.g., `LazyLoader.vue`)

## Resolution

Rename the component to avoid the `Lazy` prefix, as it is reserved by Nuxt for lazy-loading functionality.
