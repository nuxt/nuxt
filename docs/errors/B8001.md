# B8001

The Nuxt instance is not available in the current context. `useNuxt()` was called outside of a module setup function or Nuxt hook where the Nuxt context is available.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- Calling `useNuxt()` outside of a module `setup()` function or Nuxt hook
- Attempting to access the Nuxt instance in a file that runs outside the Nuxt build context

## Resolution

Use `useNuxt()` only inside a module's `setup()` function or within Nuxt hooks:

```ts
export default defineNuxtModule({
  setup(options, nuxt) {
    // nuxt is available here directly
    // or use useNuxt() inside hooks
    nuxt.hook('build:before', () => {
      const nuxt = useNuxt()
    })
  }
})
```
