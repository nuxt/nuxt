# B8002

The `createResolver()` function was called without the required `base` argument. This argument is needed to resolve paths relative to the module's location.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- Calling `createResolver()` without providing a base argument
- Forgetting to pass `import.meta.url` or a directory path

## Resolution

Pass `import.meta.url` or a directory path as the base argument:

```ts
import { createResolver } from '@nuxt/kit'

export default defineNuxtModule({
  setup () {
    const { resolve } = createResolver(import.meta.url)
    // Now you can use resolve('./runtime/plugin')
  },
})
```
