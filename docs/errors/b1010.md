# B1010

A file was modified by another process while the build cache was attempting to read it. This is typically a transient issue caused by concurrent file modifications.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- File modified by another process during build cache read
- Concurrent file system operations during build

## Resolution

This is usually transient — retry the build. If the issue persists, check for external processes modifying files during the build.
