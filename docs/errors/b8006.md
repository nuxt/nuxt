# B8006

Nuxt could not be found from the current working directory. The `nuxt` package is not installed or cannot be resolved.

::note
**This is a build error.** The build will fail when this condition is encountered.
::

## Common causes

- The `nuxt` package is not installed in the project
- Running a Nuxt command from a directory that is not a Nuxt project

## Resolution

Install Nuxt in your project:

```bash
npm install nuxt
```

Ensure you are running commands from the root of your Nuxt project.
