# B8001

`useNuxt()` was called outside of a context where the Nuxt instance is available. It only works synchronously inside a module's `setup()` function or within a Nuxt hook.

## Resolution

```ts
export default defineNuxtModule({
  setup (options, nuxt) {
    // `nuxt` is passed directly here
    nuxt.hook('build:before', () => {
      const nuxt = useNuxt() // also available inside hooks
    })
  },
})
```
