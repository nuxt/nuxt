# B3002

You are using `.server.vue` components with `ssr: false` in your Nuxt configuration. Server components require SSR or the `componentIslands` experimental feature to function correctly.

::note
**This is a build warning.** The build will continue, but you should address this issue.
::

## Common causes

- Using .server.vue components with ssr: false without enabling componentIslands
- Misunderstanding of server component requirements

## Resolution

Set `experimental.componentIslands` to `true` in your `nuxt.config` to enable server components without SSR:

```ts
export default defineNuxtConfig({
  experimental: {
    componentIslands: true
  }
})
```
